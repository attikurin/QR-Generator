教材用QRコード一括生成ツール

教材のURLや補足情報のテキストから、印刷に最適なレイアウトでQRコードカードを一括生成するWebツールです。
HTMLファイル1つで動作するため、インストール不要でブラウザがあればどこでも利用できます。

主な機能

Excelからコピペで一括作成: Excelやスプレッドシートのデータをそのまま貼り付けて、大量のQRコードを一度に生成できます。

印刷レイアウト最適化: A4用紙サイズに自動でレイアウトされます。

デザイン調整: QRコードの色、サイズ、枠線の有無などをリアルタイムで調整可能です。

PDF保存対応: ブラウザの印刷機能を使って、レイアウトを崩さずにPDFとして保存できます。

使い方

1. ファイルを開く

ダウンロードした index.html をダブルクリックして、Google ChromeやEdgeなどのブラウザで開いてください。

2. データを準備・入力

ExcelやGoogleスプレッドシートで以下の並びのデータを作成し、コピーしてください。
（※1行目は不要です。データ部分のみコピーしてください）

A列 (タイトル)

B列 (URL)

C列 (説明・任意)

数学プリントNo.1

https://www.google.com/search?q=https://example.com/math

復習用

英語リスニング

https://www.google.com/search?q=https://example.com/eng

Track 01

コピーしたデータを、ツール左側の「1. データを入力」欄に貼り付けます。
※ URLのみのリストでも動作します。

3. レイアウトとデザインの調整

左側の設定パネルで以下を調整します。

横の数・縦の数: 1ページあたりの枚数を変更します（例: 3×4=12枚）。

QRコードの色: インク節約やクラスカラーに合わせて色を変更できます。

枠線を表示: 切り取り線が必要な場合はチェックを入れます。

QRサイズ補正: QRコードが小さすぎて読み取りにくい場合などに調整します。

4. プレビュー更新

設定を変更したら、「プレビュー更新」ボタンを押して右側のプレビューを確認します。

5. 印刷 / PDF保存

「印刷 / PDF保存」ボタンを押します。

紙に印刷する場合: プリンターを選択して印刷します。

PDFにする場合: 送信先（プリンター選択）で「PDFに保存」を選択して保存します。

動作環境

最新のGoogle Chrome, Microsoft Edge, Safari, Firefox

インターネット接続（初回読み込み時にライブラリをCDNから取得するため必要です）

※オフラインで使用したい場合は、HTML内のスクリプトをローカルに保存してパスを書き換える必要があります。
